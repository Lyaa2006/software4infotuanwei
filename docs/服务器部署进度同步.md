## 2. 当前部署方式

项目当前是“前后端分离 + Nginx + Cloudflare Tunnel”的方式：

- 后端目录：`~/software4infotuanwei/server`
- 前端目录：`~/software4infotuanwei/webpage`
- 前端构建产物由 Nginx 提供
- Nginx 将 `/api`、`/uploads` 反向代理到后端 `127.0.0.1:3001`
- 外网访问当前通过 Cloudflare Quick Tunnel 暴露

当前临时外网地址：

- `https://entities-paths-bowling-concluded.trycloudflare.com`

说明：

- 这是临时 Quick Tunnel 地址，适合当前联调和展示测试
- 如果后续要长期稳定使用，应切换为 Cloudflare Named Tunnel + 自有域名

## 3. 当前启动方式

### 3.1 后端

后端环境文件位置：

- `~/software4infotuanwei/server/.env`

当前实际使用的关键配置：

```env
PORT=3001
DB_HOST=127.0.0.1
DB_PORT=5432
DB_USER=system
DB_PASSWORD=123456
DB_NAME=student_service_platform
NODE_ENV=production
PUPPETEER_EXECUTABLE_PATH=/usr/bin/chromium-browser
```

后端启动命令：

```bash
cd ~/software4infotuanwei/server
npm start
```

### 3.2 前端

前端构建命令：

```bash
cd ~/software4infotuanwei/webpage
npm install
npm run build
```

说明：

- 前端不是直接跑 Vite 开发服务器
- 当前是先构建，再由 Nginx 提供 `dist/` 静态页面

### 3.3 Tunnel

当前测试使用的是 Cloudflare Quick Tunnel，启动方式为：

```bash
cloudflared tunnel --url http://127.0.0.1:80
```

# Web 端学生标签管理说明

## 标签在哪里管理

学生标签现在统一在 Web 端的 学生标签管理 页面维护，不再在 学院公告与提醒 页面里编辑。

入口路径：/tag-management

## 如何进入

1. 登录 Web 端管理员账号。
2. 回到首页 Dashboard。
3. 在 班团与管理 分组中点击 学生标签管理。

## 如何编辑学生标签

1. 进入 学生标签管理。
2. 在学生列表中找到目标学生。
3. 点击该学生右侧的 编辑标签。
4. 在弹窗中查看学号、姓名、当前标签。
5. 输入新的标签内容后点击 保存。
6. 保存成功后，列表会立即显示更新后的标签；刷新页面后仍应保持。

点击 取消 会关闭弹窗，不会修改标签。

## 如何按标签发送通知

1. 进入 学院公告与提醒 /reminder。
2. 填写通知标题和通知内容。
3. 在发送范围中选择 按标签发送。
4. 点击可用标签 chip 进行选择，也可以在输入框中手动输入标签。
5. 点击 发送。

如果选择 按标签发送 但没有选择或填写任何标签，页面会提示 请选择至少一个标签。

## Activity 页面为什么保留骨干设置

班团活动管理需要维护班团骨干身份，用于活动提案、审核和权限判断。因此 /activity 中的 设为骨干 / 移除骨干 功能继续保留。学生标签管理负责通用标签维护，班团活动管理中的骨干设置负责活动场景下的骨干身份维护。

## 多标签输入规则

多个标签可以使用以下任意方式分隔：

- 英文逗号
- 中文逗号
- 顿号
- 空格
- 换行

重复标签会自动合并。清空输入后保存，表示该学生暂无标签。

## 常见问题

### 标签为空怎么办

可以保存为空。保存后该学生会显示 暂无标签。

### 选择标签发送通知时没选标签怎么办

如果发送范围是 按标签发送，必须至少选择一个标签或手动输入一个标签，否则无法发送。

### 保存标签失败怎么办

页面会显示错误提示。请先确认当前账号是管理员，并检查后端服务和数据库连接是否正常。

### 修改标签后 Reminder 的可用标签是否会更新

会。Reminder 页面会从当前学生标签数据中读取可用标签。修改标签后重新进入或刷新 /reminder，可用标签会更新。
